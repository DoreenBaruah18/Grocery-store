# FreshMart - Online Grocery Store

A simple web-based **Online Grocery Shopping System** developed using **HTML, CSS, and JavaScript**.

This project simulates a real-world e-commerce platform where users can browse products, filter categories, add items to cart, and complete a checkout flow.

It was built to practice frontend development, DOM manipulation, and JavaScript logic handling.

---

## Features

- User Registration & Login using LocalStorage
- Home page with product display
- Live search functionality
- Category filtering system:
  - Fruits
  - Vegetables
  - Dairy
  - Snacks
- Add to Cart system
- Cart page with total calculation
- Delivery address form
- Payment page UI simulation
- Order success page
